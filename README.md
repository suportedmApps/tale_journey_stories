# Tale Journey Content Repository

Este repositório armazena todo o conteúdo remoto utilizado pelo aplicativo Tale Journey.

## Estrutura

- stories → histórias completas com capítulos
- chapters → capítulos de séries
- quick_reads → leituras rápidas

## Idiomas suportados

- en → English
- es → Español
- fr → Français
- ja → 日本語
- pt → Português (opcional)

## Regras

- sempre manter JSON válido
- usar UTF-8
- evitar textos muito longos em quick_reads
- manter consistência de level
- nunca alterar IDs existentes
- levels padronizados:
  - Beginner
  - Basic
  - Intermediate
  - Advanced
  - Fluent
